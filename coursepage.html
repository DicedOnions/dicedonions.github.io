HTML Table Tags
Tag Description
<table> Denotes the start of an HTML table in a web page
<tr> Denotes the start of a row in an HTML table (NOTE:These tags must exist inside <table>...</table> tags in order to be valid, and work correctly)
<td> Denotes the start of a cell (or Table Data) in an HTML table (NOTE: these tages must exist inside <tr>...</tr> tags in order to be valid, and work correctly)
<th> Very similar to the <td> tags described above but the text is bold and centered.
<caption> Will place a caption on an HTML table(NOTE:These tags must be implemented right after the oprning <table> tags in order to be valid, and work correctly)
